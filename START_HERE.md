## 📞 Contact Form Setup

The form currently shows an alert. To make it work:

**Easiest Way (Netlify Forms):**
1. In `contact.html`, find line 100: `<form class="contact-form" id="contactForm">`
2. Change to: `<form class="contact-form" id="contactForm" netlify>`
3. Redeploy
4. Form submissions appear in Netlify dashboard
