## 🔄 Want to Replace Placeholder Images?

You can replace these placeholder images anytime with your own:

### Option 1: Keep Using URLs (Easiest)
Simply edit the HTML files and replace the Unsplash URLs with your own image URLs.

### Option 2: Use Local Images
1. Save your images in the `assets` folder with these names:
   - `logo.png`
   - `hero-bg.jpg`
   - `storefront.jpg`
   - `handguns.jpg`
   - `rifles.jpg`
   - `shotguns.jpg`
   - `ammunition.jpg`
   - `accessories.jpg`
   - `optics.jpg`
   - `custom-build.jpg`

2. Update the HTML files to use local paths:
   - Change `https://images.unsplash.com/...` to `assets/filename.jpg`

3. Redeploy to Netlify
